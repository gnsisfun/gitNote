You can see how the say block makes a bubble appear:

    say (args, util) {
        // @TODO in 2.0 calling say/think resets the right/left bias of the bubble
        let message = args.MESSAGE;
        if (typeof message === 'number') {
            message = parseFloat(message.toFixed(2));
        }
        message = String(message).substr(0, Scratch3LooksBlocks.SAY_BUBBLE_LIMIT);
        this.runtime.emit('SAY', util.target, 'say', message);
    }

If your extension has access to the runtime, you can do something similar, like

  sayUserAgent (args, util) {
    this.runtime.emit('SAY', util.target, 'say', navigator.userAgent)
  }
